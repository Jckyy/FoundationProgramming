About Biorythms

Like astrology and numerology, biorhythms remain a commonly used predictor of the human condition.  The following quotes explain the ideas succinctly:-

"Very briefly, biorhythm cycles consist of three natural human cycles; the physical, the emotional (or sensitivity), and the intellectual.  Each cycle has a unique period and therefore the interaction of these cycles constantly varies.  The physical cycle lasts 23 days, the emotional cycles 28 days, and the intellectual 33 days.  They all start at the zero value on the day of birth and begin the positive part of the cycle.  Half way through the cycle they again cross zero and move into the negative part of the cycle.  This cyclic nature is analogous to a wave pattern." - From Biorhythm Cycles by Chuck Anderson.

"According to believers in biorhythms, a person's life is affected by rhythmic biological cycles which affect one's ability in various domains, such as mental, physical, and emotional activity. These cycles begin at birth and oscillate in a steady sine wave fashion throughout life; thus, by modeling them mathematically, a person's level of ability in each of these domains can be predicted from day to day." - From Wikipedia: Biorhythms.

"The intellectual biorhythm lasts for 33 days and relates to your mental capabilities, creativity, aptitude to solve problems.
The emotional biorhythm last for 28 days (just like the Moon's cycle) and relates to your emotional stability or stress.
The physical biorhythm lasts for 23 days and is related to your physical energy, strength, health, stamina."
- From Bioryhthm Calculator.

Your Task

High level pseudocode:

High level pseudocode:
  1. Obtain the user's day of birth.
  2. Calculate biorhythm energy levels for the user.
  3. Tell the user about his energy levels by writing HTML to the document.

Low level pseudocode:

  1. Obtain the user's year of birth using the function prompt(). (assume return value correct)
  2. Obtain the user's month of birth as a string using prompt(). (assume return value correct)
  3. Obtain the user's day of birth using prompt(). (assume return value correct)
  4. Pass the year, month and day as three separate arguments to a constructor for Date in order to create a date object that represents the users date of birth.
  5. Construct a date object to represent the current date.
  6. Calculate the user's age in days.
  7. Calculate the intellectual, emotional and physical energy levels.
  8. Output a statement saying how many days the user has been alive, as an integer.
  9. Output heading 1 title with the text "Your biorhythm energy levels are:-"
  10. Output the energy levels as an unordered list.
    • Output the intellectual energy level using the word "intellectual": followed by a fixed format number expressed as a percentage with a single decimal place, all in the color green.
    • Output the emotional energy level using the word "emotional": followed by a fixed format number expressed as a percentage with a single decimal place, all in the color red.
    • Output the physical energy level using the word "physical": followed by a fixed format number expressed as a percentage with a single decimal place, all in the color blue.

How To

How to Calculate the user's month of birth as a number

You are required to obtain a 3 letter abbreviation of the month from the user, and you need to convert this to a number where January=0 through to December=11.

  • Take the string entered.
  • Get the substring being the first three characters.
  • Convert to lowercase.
  • Find the starting location of the three letter abbreviation in the month abbreviations string.
  • Divide this by 4.
This is not the best way to find the month number, but it allows us to practice searching in a string

How to make the date objects

 You need a date object for the user's date of birth and a date object for the current date.
  • If we construct a date with the year, month and date as arguments, then we get a date object for that date.
  • If we construct a date with no arguments then we get a date object for today.

How to calculate the user's age in days

  • Use the getTime() member function to get a Date object's time in milliseconds. You need the date of birth in milliseconds as well as the current time.
  • Subtract the milliseconds of the user's date of birth from the milliseconds of the current time to get the user's age in milliseconds.
  • Divide the user's age in milliseconds by the number of milliseconds in a day to get the user's age in days.
  • Use the Math.floor() method function to reduce the number to an integer.

How to calculate an energy level

  • Multiply the user's age in days by two times Math.PI and divide by the length in days of the appropriate cycle.
  • Use Math.sin() to take the sine of the result. The result of the sin() function call will be the biorhythm energy for that cycle.

How to show color in HTML

The correct way to apply color in HTML is to use CSS. However, we are not required to know CSS for this course, so let us do something much simpler:-
